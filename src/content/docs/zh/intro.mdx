## 介绍

这是一个基于 T3 Stack 的现代 SaaS 应用程序模板，包含以下特性：

- **Next.js 15** - 使用 App Router 的现代 React 框架
- **TypeScript** - 类型安全的开发体验
- **Tailwind CSS** - 实用优先的 CSS 框架
- **Drizzle ORM** - 类型安全的数据库 ORM
- **Better Auth** - 现代认证解决方案
- **国际化支持** - 使用 next-intl 的多语言支持

## 快速开始

1. 克隆项目
2. 安装依赖：`pnpm install`
3. 启动开发服务器：`pnpm dev`

开始构建你的下一个 SaaS 应用程序吧！
